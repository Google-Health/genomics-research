## TRIAD-HFpEF

This repository contains code for defining and training models for
**TRIAD-HFpEF**, a tri-modal framework for studying heart failures with
preserved ejection fraction (HFpEF), described in detail in this
[publication](https://doi.org/10.64898/2026.02.07.26345811).

This consists of a family of models,
TRIAD-LAB, TRIAD-CMR, and TRIAD-ECG, utilizing three different modalities
of medical data: lab measurements, cardiac magnetic resonance imaging,
and electrocardiograms.

This repository contains models for TRIAD-CMR, while the models for the other
two modalities can be found in the following repositories:

* [TRIAD-ECG](https://github.com/jackosullivanoxford/TRIAD_ECG_HFPEF/tree/main)
* [TRIAD-LAB](https://github.com/jackosullivanoxford/TRIAD_LAB_HFPEF/tree/main)
